# Git-Matlab     
!git status （代码提交更新）    
!git add （增加文件）    
!git commit -m “此处填写评论”（提交修改）    
!git pull （将远程代码pull下来，看看是否有冲突）    
!git push （若没有冲突直接将本地分支的代码push到远程分支上）           
!git remote set-url origin URL   (更换远程仓库地址，URL为新地址)
